# VDMFD

A Python tool for Linux that performs fast recursive metadata search of videos. VDMFD stands for 'Video Metadata Finder'.

## Requires

`ffprobe` which is provided by `ffmpeg`.

`xdg-mime` part of `xdg-utils`.

## Search options

-duration
-size
-filename
-container 
-bitrate
-width
-height
-framerate

## Logical operators
-a AND
-o OR
